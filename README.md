# TalkitAPI-docs
[Talkit](https://nandeyanen.ie-t.net/talkit)のAPIの仕様書です。OpenAPI3.0に準拠しています。
yamlを[こちら](https://editor.swagger.io)で読み込むとより詳細なレスポンスなどがわかります。
サーバー側の処理は手付かずなので使用はできません。

> [!WARNING]
